Copy these environment variables into Vercel:

NEXT_PUBLIC_SUPABASE_URL
[Your Supabase Project URL from .env.local]

NEXT_PUBLIC_SUPABASE_ANON_KEY
[Your Supabase Anon Key from .env.local]

SUPABASE_SERVICE_ROLE_KEY
[Your Supabase Service Role Key from .env.local]

IMPORTANT: Enable for Production, Preview, AND Development environments!

Note: Get the actual values from your .env.local file or Supabase dashboard.
