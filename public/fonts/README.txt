LAZY FOX FONT
=============

The site is set up to use a Lazy Fox–style look. Right now it uses the free
"Caveat" font (Google Fonts) so it works with no extra steps.

To switch to the real Lazy Fox font, add the font files here:

  - LazyFox.woff2   (preferred)
  - LazyFox.woff    (optional, for older browsers)

If your files have different names (e.g. LazyFox-Regular.woff2), either rename
them to LazyFox.woff2 / LazyFox.woff or update the @font-face paths in
app/globals.css.

Lazy Fox is a commercial font (not on Google Fonts). You can get it from:
  - pixelcolours.com
  - MyFonts / Envato Elements (search "Lazy Fox")

Once the files are in this folder, the site will use Lazy Fox automatically;
Caveat remains the fallback if the files are missing.
